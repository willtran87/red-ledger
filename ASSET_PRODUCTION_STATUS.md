## Source Authority

2. `art/source/keyed`: original keyed copies used to identify the intended key color.
3. `art/source/alpha`: chroma-removed intermediates.
4. `art/source/alpha/normalized`: the only source consumed by runtime sprite builders.
5. `art/source/keyed/normalized`: exact-key, evenly spaced review sheets rebuilt from normalized alpha.
6. `assets/public_runtime`: fixed-canvas, palette-locked shipping images.
